# 구현

- framer-motion을 사용한 애니메이션 효과 구현
- Axios를 이용해 firebase와 RESTful API 연동
- Vercel을 사용한 CI/CD 및 배포
- firebase의 실시간 데이터베이스 사용
- 이미지 로드 최적화에 대한 고민
- 모바일, 태블릿 반응형 구현 예정
- 상태관리 Zustand에서 Redux로 변경
- Redux Toolkit으로 콘텐츠 데이터 관리
- Custom Hook으로 비즈니스 로직, 비동기 API 호출 처리
- 반응형 웹 디자인
