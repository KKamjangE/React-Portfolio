## 프로젝트 개요

이 프로젝트는 프론트엔드 개발자 안제민의 포트폴리오 사이트입니다.

## 기술 스택

- **Frontend**: React (TypeScript)
- **style**: Tailwind CSS v4
- **UI**: Shadcn/UI
- **package manager**: pnpm
- **배포**: Vercel
- **전역 상태관리**: Zustand
- **코드 컨벤션, 코드 포멧팅, 린트**: biome
